PrepHub — Q&A Preparation Website
A full-stack Q&A preparation platform with 3D design elements, authentication, topic-based questions, and admin management — inspired by the reference screenshots.

Key Features
Feature	Description
Login/Signup	Dark themed glassmorphism login & signup pages with 3D floating card
Questions Page	Browse questions by topic with expandable answers
Admin Dashboard	Manage users (approve/revoke/delete) and questions (CRUD)
Authentication	JWT-based auth with admin role (admin/1234)
3D Design	CSS 3D transforms, parallax, floating animations, glassmorphism
Dark Theme	Deep navy/dark theme with purple accent colors
Tech Stack
Backend: Node.js + Express
Frontend: Vanilla HTML/CSS/JS (served by Express)
Storage: JSON file-based database (no external DB)
Auth: JWT tokens with bcrypt password hashing
Design: 3D CSS transforms, glassmorphism, micro-animations
Proposed Changes
Backend — server.js
Core Express server with:

JWT authentication middleware
REST API endpoints for auth, users, questions, topics
Static file serving for frontend
JSON file database read/write helpers
API Endpoints:

Method	Route	Access	Description
POST	/api/auth/login	Public	Login with username/password
POST	/api/auth/signup	Public	Register new user
GET	/api/auth/me	Auth	Get current user info
GET	/api/topics	Auth	List all topics
POST	/api/topics	Admin	Create topic
DELETE	/api/topics/:id	Admin	Delete topic
GET	/api/questions	Auth	List questions (filter by topic)
POST	/api/questions	Admin	Create question
PUT	/api/questions/:id	Admin	Update question
DELETE	/api/questions/:id	Admin	Delete question
GET	/api/users	Admin	List all users
PUT	/api/users/:id/status	Admin	Approve/revoke user
DELETE	/api/users/:id	Admin	Delete user
Database — data/db.json

Admin user (username: admin, password: hashed 1234)
